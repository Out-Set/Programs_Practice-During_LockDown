Prompt box pop-up

A prompt box is often used , if you want the user to input a value before entering a page.

When an prompt box pops-up, user will have to click either "OK" or "CANCEL"to proceed, after entering a input.

If the user clicks "OK", the box returns the Input-value.

If the user clicks "CANCEL", the box returns Null.

It is used as: promt("message to show on alert box","default value");

The 2nd(default value ) argument is optional.

It is used when we needs conermation from users.
