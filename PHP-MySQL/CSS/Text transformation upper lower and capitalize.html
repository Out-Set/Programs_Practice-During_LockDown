Text transformation

It can be done by 3-ways....

  • p.upercase{text-transform: uppercase;}
  • p.lowercase{text-transform: lowercase;}
  • p.capitalize{text-transform: capitalize;}

p.uppercase/lowercase/capitalize means: A paragraph whose group's attribute value is uppercase/lowercase/capitalize only that paragraph will be consider.
