Text Allignment

  • It is used to set the horizontal allignment of text.
  • Text can be centred, or alligned to the right or left, or justified.
  • Sample form: -selector{property} inside "HEAD TAG".
    i.e. h1{text-align:position(whatever you want);}
    After declaring it wherever it finds the text inside h1(paragraph,table,anchor,link etc...) the align property will be applied.

##Every text consumes the total area of screen,about it's width.We usually apply align properties in this areas.
