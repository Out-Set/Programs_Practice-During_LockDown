How to set border's Style/Color/Width/Radius

1.Border-Style

  • Every HTML element that consumes space on the web page, is a rectangular region.
  • This rectangular region can be bordered using CSS-properties.
  • 'border-style'-properties allows you to choose, one from the following varios style of the border.
    1. -solid
    2. -dotted
    3. -dashed
    4. -double
    5. -groove
    6. -inset
    7. -uotset

    i.e. border-style: solid; , border-style: dotted;

2.Border-Width

  • This property specifies the width of the border.

    • i.e. border-width: 5px;

3.Border-Color

  • It specifies the color of the border.

    • i.e. border-color: blue;

4.Border-Radius

  • Border-Radius makes the corner curved.

    • i.e. border-radius: 4px;

## INDIVIDUAL SIDE-METHOD

>>border-left/right/top/bottom-style/color/width/radius: value;

  • Only one should be choosen from 'left/right/top/bottom' and 'style/color/width/radius'

    • i.e. border-left-style: dashed; , border-right-color: blue; , border-top-width: 10px; , border-bottom-radius: 3px; etc.

>>border-style/color/width/radius: value(top) value(right) value(bottom) value(left);

  • Only one should be choosed from 'style/color/width/radius' , Then enter the 4-values for(top,right,bottom,left).

    • i.e. border-style: dashed solid dotted double; , border-color: blue green yrllow pink; etc.

>>border-style/color/width/radius: value(top) value(right & left both) value(bottom);

  • Only one should be choosed from 'style/color/width/radius' , Then enter the 3-values for(top,right & left,bottom).

    • i.e. border-style: dashed solid double; , border-color: blue green pink; etc.

>>border-style/color/width/radius: value(top & bottom both) value(left & right both);

  • Only one should be choosed from 'style/color/width/radius' , Then enter the 2-values for(top and bottom,left and right).

    • i.e. border-style: dashed double; , border-color: blue pink; , border-width: 2px 5px; etc.

>>border-style/color/width/radius: value(one value for all);

  • Only one should be choosed from 'style/color/width/radius' , Then enter the 2-values for(top and bottom,left and right).

    • i.e. border-style: dashed; , border-color: blue; , border-width: 5px; , border-radius: 10px; etc.
