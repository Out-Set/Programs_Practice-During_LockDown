How to change the color of text

  1. Simply writting the color name.
    -Body{color:blur;}
  2. Writting the hexadecimal code.
    -h1{color:#00ff00;} -These are actually the rgb:[a-f & 0-9]
  3. writting the RGB value.
    -h1{color:rgb(0-255,0-255,0-255);} you may any no b/w 0-255.
  4. HSL(hue,saturation,lightness)
  5. HSLA(hue,saturation,lightness,alpha)
