Internal style sheet

1.Internal style sheet: means "CSS" is included in the same HTML file.

2.We use "style attribute" to write CSS properties into "head tag" to any HTML file.

3.Type attribute of "style tag" specify the type of MIME type(text/css).

4.Internal style sheet can be used only on a specific HTML page, Not to all HTML pages.

5.Off Course...We can use an "External style sheet" into an "Internal style sheet" HTML page,through "link attribute".

#It has more preference than External-sheet but less preference than Inline-sheet.
