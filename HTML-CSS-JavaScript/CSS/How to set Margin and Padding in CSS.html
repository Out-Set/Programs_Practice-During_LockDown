How to set Margin and Padding in CSS

Every HTML pages consumes space in the web page, is a rectangular shape.

The outside area of any text's border, is Margin.

The inside area of any text's border, is Padding.

The Margin/Padding can be set as....

  • margin: 10px;
  • padding: 5px;
