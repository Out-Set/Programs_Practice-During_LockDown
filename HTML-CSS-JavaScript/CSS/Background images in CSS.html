Background images in CSS

>>Every HTML element that consumes spaces in the web page, is a rectangular shape.

>>This rectangular region can be filled with specific Images.

>>using background-image: url('image-file's name/image-file's full path);

*# By default, if image size is less than web page size, image will repeat horizontally and vertically.

To repeat only Horizontally(in X-axis only)...

  • Write: background-repeat: repeat-x;

To repeat only vertically(in Y-axis only)...

  • Write: background-repeat: repeat-y;

To stop the repetition...

  • Write: background-repeat: no-repeat;
